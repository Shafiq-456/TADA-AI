# 🚀 TADA AI – Autonomous Data Analyst & Business Intelligence Agent

![TADA AI Banner](https://via.placeholder.com/1200x300?text=TADA+AI)

## 📊 Overview

TADA AI (Autonomous Data Analyst & Business Intelligence Agent) is a full-stack AI-powered analytics platform designed to help users transform raw datasets into actionable business intelligence.

The platform enables users to upload datasets, clean and analyze data, generate interactive visualizations, forecast future trends, produce executive reports, and interact with AI-powered data analysts—all from a single unified dashboard.

---

## 🎯 Problem Statement

Organizations often struggle with:

* Manual data cleaning and preprocessing
* Complex analytics workflows
* Limited access to data science expertise
* Time-consuming report generation
* Difficulty extracting business insights from large datasets

TADA AI automates these processes using Artificial Intelligence, Machine Learning, and Business Intelligence techniques.

## 🚀 Getting Started

### Frontend Setup

```bash
cd tada-ai
npm install
npm run dev
```

Frontend URL:

```text
http://localhost:5173
```

---

### Backend Setup

```bash
cd tada-backend

python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt

uvicorn main:app --reload
```

Backend URL:

```text
http://localhost:8000
```

API Documentation:

```text
http://localhost:8000/docs
```
